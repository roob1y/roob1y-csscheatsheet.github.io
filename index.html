Background Properties

Background Description URL
background-image The background-image CSS property sets one or more background images on an element MDN doc
background-position sets the initial position for each background image. The position is relative to the position layer set by background-origin MDN doc
background-size sets the size of the element's background image. The image can be left to its natural size, stretched, or constrained to fit the available space MDN doc
background-repeat sets how background images are repeated. A background image can be repeated along the horizontal and vertical axes, or not repeated at all MDN doc
background-attachment sets whether a background image's position is fixed within the viewport, or scrolls with its containing block MDN doc
background-origin sets the background's origin: from the border start, inside the border, or inside the padding MDN doc
background-clip sets whether an element's background extends underneath its border box, padding box, or content box MDN doc
background-color sets the background color of an element MDN doc
